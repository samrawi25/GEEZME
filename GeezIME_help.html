GeezIME III – Help

1. Basics

  • Toggle GeezIME on/off: press F12.
  • Status badge: the small floating window shows current state:
    • ግ = GeezIME is ON.
    • EN with red background = GeezIME is OFF (English / normal keyboard).
    • ግ ↑ = Caps Lock is ON, ግ # = Num Lock is ON.
  • Move badge: drag it with the left mouse button.
  • Right‑click badge: opens the menu (Writer, Input Mode, Help, etc.).

2. Input modes

Right‑click the badge → Input Mode:

  • Transliteration (default): type Latin letters that are converted to Geez.
  • Amharic Layout: single Latin keys are mapped directly to common Amharic syllables.
  • Tigrinya Layout: single Latin keys mapped for Tigrinya usage.

3. Amharic / Tigrinya layout modes

In these modes, GeezIME ignores transliteration sequences and maps single keys directly:

  • Consonant keys (e.g. k, g, s, m, …) → the “a” form (4th order) of that family.
  • Vowel keys (a e i u o) → stand‑alone vowels.
  • Number row (1–0) → Ethiopic numerals ፩–፲.
  • Punctuation (.,;:) → Ethiopic punctuation.

Use these modes when you want “one key = one Geez character” without multi‑letter combinations.

4. On‑screen keyboard

  • Right‑click the badge → On‑screen Keyboard.
  • Click any Geez character: it is sent to the application that currently has the text cursor.
  • You can resize this window; buttons will adjust.

5. Geez Writer window

  • Right‑click badge → Open Writer.
  • Type Geez/Tigrinya/Amharic text with GeezIME active.
  • Use the Writer menu:
    • File → Open…: open a UTF‑8 text file.
    • File → Save As…: save your text as UTF‑8.

6. Tips

  • If a shortcut (Ctrl+C, Ctrl+V, Alt+Tab…) doesn’t behave as expected, toggle GeezIME off with F12, use the shortcut, then toggle it on again.
  • Use Transliteration mode when you want full control of every order (e.g. se/si/sa/sie/s/so).
  • Use Amharic/Tigrinya mode when you prefer a simpler, direct Geez keyboard feel.
